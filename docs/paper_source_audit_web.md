Date audited: 2026-03-04

This note records a lightweight source audit for statistical methods cited or used in the manuscript and analysis scripts.

## Audited Methods

1. Holm-Bonferroni multiple testing correction
- Primary source: Holm, 1979, *Scandinavian Journal of Statistics*.
- DOI: https://doi.org/10.2307/4615733

2. Clopper-Pearson exact binomial interval/bound
- Primary source: Clopper and Pearson, 1934, *Biometrika*.
- DOI: https://doi.org/10.1093/biomet/26.4.404

3. Bootstrap confidence intervals
- Primary source: Efron and Tibshirani, 1993, *An Introduction to the Bootstrap*.
- Publisher page: https://www.routledge.com/An-Introduction-to-the-Bootstrap/Efron-Tibshirani/p/book/9780412042317

4. Mann-Whitney U test
- Primary source: Mann and Whitney, 1947.
- DOI: https://doi.org/10.1214/aoms/1177730491

5. Miller-Madow entropy bias correction
- Primary source: Miller, 1955.
- DOI: https://doi.org/10.1002/bimj.19550070311
