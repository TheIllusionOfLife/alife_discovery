# Legacy

Archived material from the `objectless_alife` research direction, prior to the Assembly Theory pivot.

DOI: [10.5281/zenodo.18713158](https://doi.org/10.5281/zenodo.18713158)

## Contents

- `scripts/`: PR #26 follow-up analysis scripts and Zenodo publishing utilities
- `tests/`: Tests for the archived PR #26 scripts
- `paper/`: ALIFE conference paper draft (LaTeX, figures, peer reviews)
- `docs/`: Stage B/C results, reproducibility notes, reviewer response mapping
- `workflows/`: PR #26 smoke-test CI workflow (`followup-smoke.yml`)
- `data/pr26_followups/`: Published Zenodo data bundle (checksums, manifest, summary outputs)

## Notes

These files are frozen and not imported by the active `alife_discovery` package.
Do not import from `legacy/` in new code.
